# Raylib-Snake

Snake made in C99 with raylib (for fun).

## Key Bindings

| Key | Function     |
|-----|--------------|
| w   | Move Up      |
| a   | Move Left    |
| s   | Move Down    |
| d   | Move Right   |
| r   | Restart Game |
| v   | Toggle VSync |
| f   | Toggle FPS   |

## Build

To Build run the following.
```
cmake -B build
cmake --build build
```
