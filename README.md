# Raylib-Snake

Snake made in C99 with raylib (for fun).

## Key Bindings

| Key | Function     |
|-----|--------------|
| w   | Move Up      |
| a   | Move Left    |
| s   | Move Down    |
| d   | Move Right   |
| r   | Restart Game |
| v   | Toggle VSync |
| f   | Toggle FPS   |

## Build

Requires Raylib 4.0.0.

Run `make` to build. Note; build is setup for MacOS only. You should be able to build on other operating sysems with some tweaks.
